# imersao-alura-dani
GlowBeauty AI ✨  O GlowBeauty AI é um projeto que desenvolvi como aluna durante a Imersão Alura. A ideia é oferecer uma ferramenta simples que ajuda o usuário a encontrar produtos de beleza de forma rápida e organizada.  Durante o processo, contei com o apoio dos professores da imersão e utilizei o Google Gemini para me auxiliar em pesquisas, estruturação de conteúdo e entendimento técnico. O objetivo foi transformar uma ideia básica em uma aplicação funcional, aplicando o que aprendi nas aulas.  O funcionamento é direto: o usuário informa suas preferências e necessidades, e a ferramenta sugere produtos relacionados, facilitando a busca. O GlowBeauty AI não pretende revolucionar nada — é apenas um projeto de estudo que representa minha evolução prática com código, conceitos de IA e criação de interfaces voltadas para o cotidiano.
